This repo is for practising how to build ESP32 based device/SW-project.
The goal is to create a device which measures temperature/humidity and then reports sensor data by bluetooth.

The plan is to split this project into several phases:
1. Create basic functionality for measuring/reporting by using classic BT and read the data by android phone by using bt terminal <DONE>
2. Create the same functionality with low energy bt for more optimized battery usage. <Not DONE>
3. Extend existing functionality by adding a screen into device. <Not DONE>
4. Create an android application for checking the temp/humidity. <Not DONE>
